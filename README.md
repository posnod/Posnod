# Project Template Posnod

Repo ini berisi template dasar untuk tim:

- **Frontend:** Next.js + Tailwind + shadcn (`frontend/`)
- **Backend:** Laravel + PostgreSQL (`backend/`)
- **Panduan Tim:** Lihat `docs/team-guidelines.md` untuk workflow, naming, testing, linting
- **Testing:** `tests/frontend/` dan `tests/backend/`

- **Dev Environment:** Gunakan `docker-compose.yml` jika ingin setup dev environment cepat

## Frontend (quick start)

1. npm install
2. npm run dev

### Struktur
- `src/app` → pages & layout
- `src/components` → UI components
- `src/hooks` → custom hooks
- `src/utils` → helpers
- `src/styles` → global css
